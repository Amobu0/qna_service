# Q&A 게시판

#### 프로젝트 소개 
 - 스프링부트 Q&A 게시판 만들기

### 주요 기능
- 질문 게시글 & 답변 CRUD

### 기술 스택
 - Frontend: Thymeleaf 
 - Backend: Spring Boot / Spring Data JPA / Java17
 - Database: MariaDB
 - Design: BootStrap

### 업데이트 내용
1) MariaDB 연결 & Answer, Question 엔티티 & Repository 생성
2) 1)에 대한 INSERT, SELECT, UPDATE, DELETE JUNIT TEST
3) 타임리프를 적용하여 질문 리스트 & 상세보기 페이지 구현
4) 답변등록 구현, 부트스트랩 적용
